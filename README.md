# Arive TPI Dashboard

GitHub Pages-ready static dashboard.

## Upload
Upload `index.html`, `README.md`, and the `data` folder to your repo root.

## Enable Pages
Settings -> Pages -> Deploy from branch -> main -> / root.

Note: GitHub Pages is public if the repository is public.
