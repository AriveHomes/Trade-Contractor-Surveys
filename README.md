# Arive TPI Dashboard

Static dashboard preview for the Arive Trade Performance Index.

## GitHub Pages setup

1. Create a new GitHub repository, for example `arive-tpi-dashboard`.
2. Upload everything in this folder to the repository.
3. Go to **Settings > Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/root` folder.
6. Save.
7. GitHub will generate a public link similar to:
   `https://YOUR-USERNAME.github.io/arive-tpi-dashboard/`

## Updating the dashboard

Replace `index.html` with the updated dashboard file whenever new survey results are processed.
